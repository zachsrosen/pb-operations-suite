PB Operations Suite

Complete Guide for Owners, Project Managers & Operations Teams

https://pb-operations-suite.vercel.app

The PB Operations Suite provides real-time visibility into your solar project pipeline, powered by live data from HubSpot. All dashboards auto-refresh every 5 minutes and are read-only (they do not modify HubSpot data).

Quick Reference - Which Dashboard Do I Use?

Dashboard Best For Use When You Need To...

Master Scheduler Scheduling Schedule site surveys, installations, and inspections
At-Risk Projects Problem Solving Find overdue, stalled, or blocked projects
PE Dashboard PE Compliance Track Participate Energy milestones and deadlines
Executive Summary Leadership See high-level KPIs, charts, and trends

Operations Dashboards

Command Center PRIMARY

Purpose: Central operations hub for daily pipeline management. This is your go-to dashboard for a complete picture of all active projects.

Key Features:

  • Pipeline overview with total projects, RTB count, PE projects, and pipeline value
  • Revenue tracking by stage and location
  • Capacity planning by crew and location
  • PE reporting with export capabilities (Excel, CSV)
  • Alerts tab for overdue milestones

How to Use:

  • Use the tabs at the top to switch views: Pipeline, Revenue, Capacity, PE, Alerts
  • Filter by location using the dropdown menu
  • Filter by PE status: All / PE Only / Non-PE
  • Search for specific projects by name or AHJ
  • Click any project row to open it directly in HubSpot
/dashboards/pb-unified-command-center.html

Master Scheduler SCHEDULING

Purpose: Schedule site surveys, installations, and inspections with visual crew management and calendar views.

Key Features:

  • Three views: Month Calendar, Week Grid, Gantt Timeline
  • Drag-and-drop scheduling for projects
  • Stage filtering: Survey, RTB, Blocked, Construction, Inspection
  • Crew capacity tracking and conflict detection
  • Auto-optimize by revenue priority
  • Weekend detection (auto-adjusts to next business day)

How to Use:

  • Select a view using tabs: Month / Week / Gantt
  • Filter the project queue by stage using the stage tabs on the left
  • Drag projects from the left queue onto calendar dates
  • Set duration and assign crew in the scheduling modal
  • Use "Auto-Optimize" to schedule all RTB projects by revenue priority

Keyboard Shortcuts: 1 Month, 2 Week, 3 Gantt, Ctrl+O Optimize, Ctrl+E Export

/dashboards/pb-master-scheduler-v3.html

Pipeline Optimizer ANALYTICS

Purpose: AI-powered bottleneck detection and priority-based scheduling recommendations.

Key Features:

  • Automatic bottleneck identification across pipeline stages
  • Priority scoring based on revenue, PE status, and urgency
  • Location efficiency analysis
  • Optimized schedule generation

How to Use:

  • Review the bottleneck analysis at the top to see where projects are stuck
  • Check the priority queue showing the top 10 highest-priority projects
  • Click "Generate Optimized Schedule" to auto-schedule RTB projects
  • Export the optimized schedule as CSV for sharing
/dashboards/pb-optimization-dashboard.html

At-Risk Projects ALERTS

Purpose: Real-time alerts for projects that need immediate attention due to missed deadlines or stalled progress.

Key Features:

  • Risk scoring algorithm with Critical vs Warning severity levels
  • Filter by risk type and location
  • Revenue at risk calculations
  • Direct links to HubSpot records for quick action

Risk Types Monitored:

  • Install Overdue - Past forecasted installation date
  • Inspection Overdue - Past inspection deadline
  • PTO Overdue - Past PTO deadline
  • Stalled - No stage movement in 30+ days
  • Blocked - Projects in RTB-Blocked or On-Hold status

Common Features Across All Dashboards

Live Data: All dashboards connect to HubSpot and auto-refresh every 5 minutes. No manual refresh needed.
  • Direct HubSpot Links: Click any project name or row to open the deal directly in HubSpot
  • Multi-Location Support: Filter and compare data across all 5 locations
  • Export Options: CSV, Excel, and clipboard exports are available on most dashboards
  • Universal Color Coding:
    • Red = Overdue / Critical / Needs immediate attention
    • Yellow = Warning / Due Soon / Approaching deadline
    • Green = On Track / Healthy / Complete
  • Read-Only: These dashboards display data from HubSpot but do not modify it. All changes must be made in HubSpot directly.

Getting Started - Recommended Workflow

  1. Start your day with the Command Center to get a complete pipeline overview
  2. Check the At-Risk Projects dashboard to identify problems needing immediate attention
  3. Schedule work using the Master Scheduler - drag and drop projects onto calendar dates
  4. Monitor PE compliance with the PE Dashboard to ensure Participate Energy deadlines are met
  5. Share the Executive Summary with leadership for weekly or monthly KPI reviews
  6. Use Mobile Dashboard when in the field for quick project lookups
Pro Tip: Bookmark the Command Center as your home page for quick daily access. Use the stage tabs to quickly focus on RTB projects ready for scheduling.
