# SEG Solar - Module Reference

## SEG-440-BTD-BG (Yukon N Series)

440W N-Type Bifacial Black Module | 108 Half-Cut Cells

### Product Overview

- **Series:** Yukon N
- **Model:** SEG-440-BTD-BG
- **Wattage:** 440W
- **Cell Technology:** N-Type Monocrystalline, Bifacial
- **Cell Count:** 108 (54 x 2 half-cut)
- **Cell Size:** 182mm
- **Glass:** Dual glass -- 2.0mm AR-coated semi-tempered front and rear
- **Frame:** Black anodized aluminum alloy
- **Connector:** MC4
- **Max System Voltage:** 1500 VDC
- **Series Fuse Rating:** 30A
