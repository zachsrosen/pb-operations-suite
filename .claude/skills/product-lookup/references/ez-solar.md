## JB-1.2 Rooftop PV Junction Box

Weatherproof junction box for DC wire transitions on rooftop solar installations. Provides a code-compliant enclosure where PV wire (USE-2/PV wire from module strings) transitions to THHN conductors routed through conduit to the inverter or combiner. Mounted directly on asphalt/composition shingle roofs.

### Specifications

| Attribute | Value |
|-----------|-------|
| Max voltage | 1,000 V DC |
| Max current | 80 A |
| Wire size range | 14 AWG to 6 AWG |
| Enclosure rating | NEMA Type 3R (rain-tight) |
| Roof slope range | 2.5:1 to 12:12 |
| Max fitting size (sidewall) | 1 inch |
| Max fitting size (floor) | 1 inch |

| Operating temp | -35 C to +75 C |

| Certifications | UL 1741, CSA C22.2 No. 290 |
| Warranty | 25 years |

### Installation Rules

- Minimum 1/2 inch spacing between uninsulated live parts and fittings (conduit, armored cable, opposite-polarity conductors)
- 3 sidewalls available for wire entry -- plan conduit routing accordingly
- Mounts on roof deck; flashing and sealant required per standard rooftop penetration practice
- All wire transitions (PV-wire to THHN) happen inside the box

### PB-Specific Usage

- Used on the PV-4 conductor schedule as the **transition point** between rooftop PV wire and conduit runs
- Appears on planset electrical one-line as the junction box between string wiring and conduit to inverter
- Typically one JB-1.2 per array section (one per roof plane or one per string home-run location)
- On Tesla inverter jobs: PV wire runs from modules through JB-1.2, transitions to THHN in conduit down to the Tesla inverter

### Related Product

**JB-1.XL** -- larger version for higher conductor counts or commercial applications. Same NEMA 3R rating, larger enclosure footprint. Use when JB-1.2 fitting count is insufficient.
