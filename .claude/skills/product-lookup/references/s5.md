## ProteaBracket (Trapezoidal Metal Roof Attachment)

Adjustable attachment bracket for mounting solar racking on trapezoidal (exposed-fastener) metal roofs. Replaces IronRidge L-Foot/HUG-based attachment on metal roof jobs. No roof penetration through the weather surface -- screws into the raised rib crown only, preserving waterproof integrity.

### Specifications

| Attribute | Value |
|-----------|-------|
| Compatible roof profiles | Most trapezoidal / exposed-fastener metal roofs |
| Max rib height | 3 inches |
| Material (standard) | Aluminum with stainless steel hardware |
| Material (heavy-duty) | A2 stainless steel bimetal |
| Fasteners | Stainless-capped screws (included) or Bulb-Tite rivets (sold separately) |
| Pre-punched holes | 4 (all must be used for rated load) |
| Sealant | Factory-applied adhesive rubber sealant |
| Rail compatibility | Any rail system L-foot or S-5! PVKIT for rail-less |
| Warranty | Lifetime (S-5! standard) |

### Installation Rules

- Mount on the **crown (top) of the trapezoidal rib** -- never in the valley/pan
- All 4 pre-punched holes must be fastened to achieve tested strength
- Factory sealant provides weatherproofing -- no additional sealant needed at screw points
- Compatible with intermediate insulation layers between roof panels and structure
- No lag screws or structural roof penetrations required
- Does not void metal roof warranty (non-penetrating to weather surface)

### PB-Specific Usage

- **Trigger**: Planset roof type shows "TRAPEZOIDAL METAL ROOF" or similar exposed-fastener metal profile
- **Replaces**: IronRidge HUG (Hugger) attachment that is standard on comp shingle jobs
- **Paired with**: XR100 rails (not XR10) -- metal roof jobs use the heavier rail
- **Racking note**: On metal roof jobs, S-5! ProteaBracket + XR100 rails replace the standard IronRidge FlashFoot2 + XR10 racking package
- When ProteaBracket is used, **no lag screws** appear on the BOM (vs. comp shingle jobs that use 5/16" x 4" lags)

## Standing Seam Clamps

For **standing seam** metal roofs (folded/crimped seam, not exposed-fastener), S-5! offers clamp-on attachments that grip the seam mechanically with setscrews. These do not pierce or penetrate the seam -- they dimple it slightly for grip.

### Common Clamp Models

| Model | Seam Type | Notes |
|-------|-----------|-------|
| S-5-S | Snap-lock standing seam | Most common residential standing seam |
| S-5-U | Universal / wide range | Fits multiple seam profiles |
| S-5-N | Narrow / specific profiles | For thinner seam geometries |
| S-5-E | Exposed fastener profiles | Select profiles |

### Clamp Sizing

- **Standard (two-setscrew)**: Heavy-duty applications (snow retention, large arrays)
- **Mini (one-setscrew)**: Lighter-duty / cost-effective for solar panel mounting

### Clamp + Solar Accessories

| Accessory | Purpose |
|-----------|---------|
| PVKIT | Direct rail-less module mounting on clamp |
| SolarFoot | L-foot platform for rail-mounted systems on exposed-fastener roofs |
| CorruBracket | Corrugated metal roof attachment (R-panel, AG-panel) |
| CorruBracket 100T / 500T | Trapezoidal corrugated profiles |

### PB-Specific Usage (Standing Seam)

- Standing seam clamp model depends on the specific seam profile -- must match the roof manufacturer's seam geometry
- PB typically uses S-5-S or S-5-U clamps when the roof is standing seam (not trapezoidal)
- Paired with XR100 rails for metal roof jobs
- Clamp selection is a design-time decision based on the metal roof spec sheet or field measurement

## ProteaBracket vs. Standing Seam Clamp Decision

| Roof Type | S-5! Product | Rail |
|-----------|-------------|------|
| Trapezoidal (exposed fastener) | ProteaBracket | XR100 |
| Standing seam (folded/crimped) | S-5-S / S-5-U clamp | XR100 |
| Corrugated (R-panel, AG-panel) | CorruBracket + SolarFoot | XR100 |
| Comp shingle (asphalt) | N/A -- use IronRidge FlashFoot2 | XR10 |
