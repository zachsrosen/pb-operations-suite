HubSpot Workflow Reference

Live catalog generated 2026-06-21 from the HubSpot Automation v4 API. 933 flows total — 736 ON, 197 OFF.

Reading the numbers: 933 raw flows is far larger than the ~150–200 logical processes PB actually runs. The inflation comes from:
  • Clone fan-out (57 flows): HubSpot can't re-enroll an object that already passed through a workflow, so revision lanes are cloned and suffixed (#1) (#2) (#3) (#4). Each clone is one logical step — this reference collapses them into a single row labeled "cloned ×N".
  • Date-stamp plumbing (59 flows): one tiny flow per "stamp <property> with today's date". Summarized as one family below.
  • Autosave noise (17 flows): Unnamed workflow - <timestamp> abandoned edits — cleanup candidates, not real automation.
